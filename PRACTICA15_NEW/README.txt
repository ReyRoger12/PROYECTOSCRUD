Sistema de Catálogos - Hamburguesas El Negrito
===============================================

Este es un sistema de escritorio desarrollado en Python utilizando la librería Flet. El sistema permite gestionar catálogos de:
- Categorías
- Artículos
- Clientes
- Empleados

Características:
---------------
- Interfaz moderna con navegación lateral.
- Conexión a base de datos MariaDB/MySQL.
- CRUD completo para cada entidad.
- Relación entre artículos y categorías.

Requisitos:
-----------
- Python 3.9+
- Librerías:
  - flet
  - mysql-connector-python
- MariaDB o MySQL configurado y con las tablas requeridas.
